LiteTran

About

LiteTran is tiny program for text translate.

It build with Qt5

LiteTran can:

  • Grab selected text by key press and show translation result in tiny tooltip (Alt+T by default)
  • Reverse translate selected text from result language to source language (Alt+R by default)
  • Display translation variants for several words (configurable)
  • Display only necessary languages (configurable)
  • Pronounce text
  • Hide in tray
